# Language Translator API

A free, fast, and reliable Language Translation API built with FastAPI and Google Translate.

## Features

- 🌍 **Multi-language support**: 80+ languages available
- ⚡ **Fast translations**: Powered by Google Translate
- 🔍 **Language detection**: Auto-detect source language
- 📝 **RESTful API**: Clean and intuitive endpoints
- 🛡️ **Rate limiting**: 30 requests per minute
- 📚 **Interactive docs**: Swagger UI and ReDoc
- 🔒 **CORS enabled**: Use from any web application

## Tech Stack

- **FastAPI**: Modern web framework
- **Google Translate**: Translation engine
- **Pydantic**: Data validation
- **Python 3.9+**: Core language
